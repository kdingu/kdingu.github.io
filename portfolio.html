Database Migration

This is the project i delivered for my bachelor thesis.
The idea was to create a program that would take all the existing data from a relational database and modify it without changing the data logic to then export it to a MongoDB NoSQL database. A dummy SQL database was used to provide the model of a working relational database and from that was created a new data model that fit best with NoSQL database technology.

Technologies
  • Java
  • SQL Server
  • MongoDB

You can take a look at the code itself on the github repo.
